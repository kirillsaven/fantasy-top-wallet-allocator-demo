# Sanitization Notes

This repository is designed to be safe for public portfolio use.

## Removed

- real wallet addresses;
- private API keys, sessions, cookies, or tokens;
- server IP addresses and deployment paths;
- raw production databases;
- real market snapshots;
- logs and generated reports;
- exact private trading logic;
- user-specific aliases or account identifiers.

## Replaced with synthetic examples

- card handles;
- projected scores;
- market prices;
- league rules;
- portfolio composition;
- strategy output.

## Why not publish the private repo directly

The private project mixed product code with runtime context, operating notes,
server topology, and historical artifacts. Publishing it as-is would create
unnecessary privacy and security risk. A focused case-study repository is a
cleaner signal for recruiters and clients.
